# 墨墨记忆卡「内容语法」说明

让记忆变简单，更高效抗遗忘。

——墨墨之使命

墨墨记忆卡 2\.0\.00 新增了三个功能：

1. **标记重点**：可以**给文字加粗**、调整文字颜色、附加背景颜色

2. **填空题**：可以在一行内，把几个连续的字`盖起来`，可以在学习的时候`翻开查看`

3. **选择题**：可以录入选择题，分为单选和多选，学习的时候可以交互，选项可以随机出现

## 功能一：标记重点

![Image](https://internal-api-drive-stream.feishu.cn/space/api/box/stream/download/authcode/?code=ZWU1ZGM0ZTNmOGY3MjdlODBlM2Q1NmNlYmVlMmNlZTJfMjI5NWU5N2VmNzM1MjFhYzY2YTdkNGJmNmE4YTgzNzZfSUQ6NzAzNjIzOTQ4NzgzNTQ4ODI1OF8xNzc5ODYyMzU5OjE3Nzk5NDg3NTlfVjM)

|![Image](https://internal-api-drive-stream.feishu.cn/space/api/box/stream/download/authcode/?code=MzkxZGIxNzBmNzIzYWUxOGI4NTQyNGY4ODNlY2I2ODVfNDNjNTIyYzMxYTcyMTcxZTlhY2RiZmJjMDdkM2I0YzZfSUQ6NzAzNjI0MDQxODU2NDA0Njg0OV8xNzc5ODYyMzU5OjE3Nzk5NDg3NTlfVjM)|- 新建卡片后，在编辑页面的左下角，可以看到“高亮”按钮，点击后键盘收起，高亮辅助区域显示。（再次点击高亮按钮或者点击右侧键盘按钮，键盘会再次弹出）<br>- 此时如果你点击高亮辅助区域的任意按钮，会在光标处插入我们的高亮语法，比如加粗是`\[T\#B\#\]`，红色文字是`\[T\#\!329e26\#\]`，只要在第二个井号后面输入你想要高亮的内容比如`\[T\#B\#给文字加粗\]`，输入后点击右上角的“预览按钮”，就可以看到自己高亮的效果<br>- 当然，你还可以框选一段文字后，点击高亮辅助按钮。比如我可以框选`给文字加粗`，然后点击“B按钮（加粗按钮）”，框选的位置就会变为`\[T\#B\#给文字加粗\]`，点击右上角的“预览按钮”，就可以看到自己高亮的效果<br>---<br>- 进阶一：如果你的高亮超过一行，每行的内容需要单独框选并点击，我们的语法不支持跨行<br>- 进阶二：如果我想加粗套着红色怎么办，语法的第一个井号前是参数，加粗是`B`，红色文字是`\!329e26`，**加粗套红色**就是：`\[T\#\!329e26,B\#加粗套红色\]`，参数不分先后用英文逗号隔开（背景是两个英文叹号，文字是一个英文叹号）<br>- 进阶三：下面举一个复杂的例子：<br>显示：我**爱****北京****天安门**<br>录入：我\[T\#B\#爱\]\[T\#B,\!c6413a,\!\!c5f1c0\#北京\]\[T\#B,\!\!c5f1c0\#天安门\]<br>- 但实际上，，，你不会需要叠加的，虽然告诉你我们支持|

|![Image](https://internal-api-drive-stream.feishu.cn/space/api/box/stream/download/authcode/?code=NDA5ODQzOTZjZWNkMDhhYjRhNjkzMGYyMDI3ZGNlMzhfOTQ2N2I3Nzg4NmE3YTZhNGE3Zjg2NDJjN2YyMDMzYThfSUQ6NzAzNjI0MDMwNjgwMjcyMDc3MV8xNzc5ODYyMzU5OjE3Nzk5NDg3NTlfVjM)|- 新建卡片后，在编辑页面的右下角，可以看到“选择题”按钮，点击后会在光标处插入我们的选择题语法`\[Choice\#\#\]`，点击右上角的“预览按钮”，就可以看到自己选择题的效果<br>- 当然，你还可以框选一段文字后，点击选择题按钮<br>- 选择题语法内只有选项，没有“题干”，所谓题干，其实就是“选择题”前面的普通文字<br>- 选择题的参数区域，支持两个参数：代表多选题的 multi、代表选项打乱的 random<br>- 选择题的正文部分，每行一个选项，星号空格开头的行是正确选项，减号空格开头的行是错误选项<br>---<br>- 进阶一：选项内支持“高亮”，但不支持填空题，主要是交互冲突，而且没有使用场景<br>- 你用得到的冷知识：答对不会奖励马克的 [🤣](https://emojipedia.org/rolling-on-the-floor-laughing/) ，你收获知识、变得博学而敏思，是最大的奖励<br>- 你用不到的冷知识一：没有星号空格或减号空格开头的行不是选项，会合并入上一行。如果在第一个选项前有不是选项的行，那些文字不会显示出来<br>- 你用不到的冷知识二：单选题有且只有一个正确选项，多选题至少有一个正确选项，否则会无法正常的显示出来<br><br>|

## 补充语法：选择题 2\.0

> 选择题在 3\.8\.00 以后可以解析新的参数 ans 了，可以无视选项内的前导字符，直接指定答案
> 
> 

```Markdown
[Choice#ans/ABC#
- A 选项
- B 选项
- C 选项
- D 选项
]
```

> 以下是一个具体的例子
> 
> 

```Markdown
下午茶是什么？
[Choice#ans/A#
- 墨墨记忆卡 CEO
- 应急食品
- 宠物而已
- 一种传统活动
]
```

## 补充语法：元素

- 行内

    - 挖空：刚刚提到了，是功能二

    - 发音：\[Audio\#ID/3636,A\#我爱北京天安门\]

        - ID 表示的是发音文件ID，

        - A参数表示遇到该发音元素后他会立刻自动发音。

    - 引用：\[Card\#ID/7e01\-7e02\-7e01\#查拉图斯特拉如是说\]

        - ID 表示的是目标卡片ID，一个引用可以挂载多个卡片，ID用减号分开

- 整行：

    - 答案线：\-\-\-

        - 最基础最经典的语法，建议每一个小伙伴熟练使用

    - 选择题：刚刚提到了，是功能三

    - 图片：\[Pic\#ID/3636,MID/2828\#\]

        - ID 表示的是图片ID

        - MID 表示的是图片遮罩文件的ID

        - 一行可以排多个图片，如果有必要的话：\[Pic\#ID/3636\#\]\[Pic\#ID/3637\#\]

    - 公式：\[E\#\#E=mc^2\]

        - 双井号后面跟的是 [Katex 语法](https://katex.org/docs/supported.html)，客户端 3\.8\.00 已经完全支持

## 补充语法：样式

### 整行文字样式

> 该功能在 2\.9\.00 以上支持解析，在官网的**可视化编辑器**可以快速编辑，接下来讲解语法
> 
> 

\[P\#H1,center,L\#哈哈\]

> 整行样式内可以嵌套行内样式和行内元素：高亮、挖空、发音、引用
> 
> 

\[P\#H1,center,L\#\[T\#B\#加粗\]\[T\#\!c6413a\#红色\]\[Audio\#ID/3636\#发音\]\[Card\#ID/7e01\#引用\]\]

### 行内文字样式

> 颜色代码只支持小写、up/down 不能同时出现
> 
> 

\[T\#B,U,I,D,up,\!ffffff\#哈哈\] \[T\#down,\!\!ffffff\#吼吼\]
